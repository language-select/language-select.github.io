<language-select> - Proof of Concept

The Web Component displays the button in the top rigth (or whereever)

Loading UI and SVG Flags Web Components dependencies only AFTER click

That means dependencies are only every loaded when required.

Delay/performance measured in console. Is it an acceptable delay

Additional Web Components are removed from the DOM again, but definition remains in memory so dependencies are loaded once

Use

Load with

<script src="https://language-select.github.io/button.min.js"></script>

Use: 

<language-select selected="nl" languages="en,nl,es,pt,..."</language-select>

Language codes are ISO-Alpha3 country codes

Style with ::part (see Page Source)

Language selection sends a "language-select" Event up the DOM tree (see console)
