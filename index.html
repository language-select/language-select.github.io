<language-select> - Proof of Concept

Load with

<script src="https://language-select.github.io/button.min.js"></script>

Use: 

<language-select selected="nl" languages="en,nl,es,pt,..."</language-select>

Language codes are ISO-Alpha3 country codes

Style with ::part (see Page Source)

Minimal UI is loaded

Loading UI and SVG Flags dependencies only AFTER click (performance measured in console)

Language selection sends a language-selected Event up the DOM tree (see console)

Additional Web Components are removed from the DOM again, but definition remains in memory so dependencies are load once
